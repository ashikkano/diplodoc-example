# Ton Rate

Возвращает актуальный курс TON в USDT

### HTTP запрос
```
GET https://api.giftstat.app/current/ton-rate
```

### Структура ответа

```json
{
    "data": [],
    "meta": {
        "limit": 200, 
        "offset": 0,
        "count": 0,
        "api_path": "/current/ton-rate",
    }
}
```
