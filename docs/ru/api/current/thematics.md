# Thematics

Возвращает актуальный список тематиеских подборок

### HTTP запрос
```
GET https://api.giftstat.app/current/thematics
```

### Структура ответа

```json
{
    "data": [],
    "meta": {
        "limit": 200, 
        "offset": 0,
        "count": 0,
        "api_path": "/current/thematics",
    }
}
```
